webapp 'typro'

For my course "Programming 1", we had to develop a website or web app with the use of HTML, CSS and Javascript/jQuery. My team - Jonas Kuske, Max M. Schneider and me - decided to let our marketing concept for the app "typro" become reality. We could translate almost all the ideas from the concept into the web app.
this is the most complex web project that i worked on so far. we built a single-page-application with jQuery mobile and also worked with canvases and html media to make camera-access possible.
unfortunately the main point of the concept, the actual analysis of fonts, is not working. Right now, each picture the user takes is assigned a random font. In the future, we want to try to implement this part with the help of open-source software.
