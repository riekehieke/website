web app 'typro'

A jQuery mobile web app based on this marketing concept. The main idea of this concept was that this web app should be able to recognize fonts via camera and identify them. This was not realizable at this stage. Now, the web application can access the camera or camera roll and save pictures linked to different user accounts. It also has a catalogue with different fonts. The single-page-application was developed with the use of HTML, CSS and of course Javascript/jQuery. Maybe in the future, me and the rest of this team, Jonas Kuske and Max M. Schneider, will further develop this project.
